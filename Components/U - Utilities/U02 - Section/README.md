# U02 - Section
v1.0.3

### Modifiers
* `section`
    * `section_y_*`: Top/bottom padding
        * `section_y_sm`
        * `section_y_md`
        * `section_y_lg`
        * `section_y_none`
    * `section_top_*`: Top padding
        * `section_top_sm`
        * `section_top_md`
        * `section_top_lg`
        * `section_top_none`
    * `section_bottom_*`: Bottom padding
        * `section_bottom_sm`
        * `section_bottom_md`
        * `section_bottom_lg`
        * `section_bottom_none`
    * `section_x_*`: Left/right padding
        * `section_x_sm`
        * `section_x_none`
* `section__content`
    * `section__content_w_*`: Maximum content width
        * `section__content_w_sm`
        * `section__content_w_md`
        * `section__content_w_full`
