# T03 - Tile 3
v1.0.0

### Dependencies
* U01 - Background Image
* U03 - Text
* U04 - Title
* U05 - Button

### Modifiers
* `tile-3`
    * `tile-3_y_*`: Top/bottom padding
        * `tile-3_y_lg`
    * `tile-3_no-box`: No "box" arround content
* `tile-3__item`
    * `tile-3__item_grow`: Grow to fill available height
    * `tile-3__item_top_*`: Top margin
        * `tile-3__item_top_sm`
        * `tile-3__item_top_lg`
        * `tile-3__item_top_none`
