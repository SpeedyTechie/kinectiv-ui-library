# T01 - Tile 1
v1.0.1

### Dependencies
* U01 - Background Image
* U03 - Text
* U04 - Title
* U05 - Button

### Modifiers
* `tile-1__part`
    * `tile-1__part_grow`: Grow to fill available height
* `tile-1__image`
    * `tile-1__image_h_*`: Image (minimum) height
        * `tile-1__image_h_sm`
        * `tile-1__image_h_md`
        * `tile-1__image_h_lg`
        * `tile-1__image_h_xl`
* `tile-1__item`
    * `tile-1__item_grow`: Grow to fill available height
    * `tile-1__item_top_*`: Top margin
        * `tile-1__item_top_sm`
        * `tile-1__item_top_md`
        * `tile-1__item_top_lg`
        * `tile-1__item_top_none`
* `tile-1__text`
    * `tile-1__text_alt-color`: Alternate color
